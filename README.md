# dc 模拟器

SeaDice JS 经济娱乐插件。

## 安装

把 `dc模拟器.js` 放进海豹骰的 `data/default/scripts` 目录，然后在海豹里重载 JS 插件或重启海豹。

插件扩展 ID 保持为 `slot-economy`，用于兼容旧版钱包、低保、奖池和排行榜存储。

## 主要功能

- 个人全局钱包：`.余额`
- 每日低保：`.低保`
- 打工：`.打工`、`.打工开始`、`.打工结束`
- 互相转账：`.转账 QQ:123456 50`、`.转账 @某人 50`
- 管理员发钱：`.加钱 QQ:123456 100`、`.加钱 @某人 100`
- 一键共产：`.一键共产`，二次确认后清空已记录钱包余额并充公
- 财富榜：`.财富榜`、`.余额榜`、`.钱包榜`
- 老虎机：`.老虎机`、`.老虎机 50`、`.老虎机50`
- 大爷幸运大转盘：`.抽奖`、`.抽奖 十连`、`.十连抽奖`
- 时髦值：`.时髦值`、`.时髦榜`
- 骰子买大小：`.买大小`、`.买大 50`、`.买小50`
- 道具俄罗斯轮盘：`.轮盘 单人 50`、`.轮盘 多人 50`、`.rr fire`
- 21 点个人局：`.21点 50`、`.bj 50`

## 当前说明

- `.加钱`、`.发钱`、`.一键共产` 需要海豹管理员权限，权限等级不低于 `100`。
- 一键共产确认成功后会重置当天低保领取次数。
- 私聊老虎机会使用独立私聊奖池，群聊老虎机使用本群奖池。
- 私聊轮盘只支持单人模式，多人轮盘只在群聊中使用。
- 21 点按“每群每人一局”保存状态。系统固定做庄，玩家之间互不等待。

## 俄罗斯轮盘规则与命令

- `.轮盘 单人 50` / `.rr solo 50`：和骰娘开启单人道具轮盘。玩家下注，骰娘跟同额虚拟注。
- `.轮盘 继续` / `.rr fire`：玩家扣动道具扳机。安全后可继续，也可换骰娘行动。
- `.轮盘 换人` / `.rr pass`：骰娘扣动道具扳机。骰娘安全后交回玩家。
- 单人局使用 6 格道具轮盘，1 格失败格。开局只旋转一次，之后连续推进。
- 单人局由最后幸存者拿走奖池。玩家获胜时得到 `下注 * 2`，玩家失败时失去下注。
- `.轮盘 多人 50` / `.rr multi 50`：开启本群多人报名，每人同额下注进入奖池。多人局也只在开局旋转一次。
- `.轮盘 加入` / `.rr join`：加入当前多人轮盘报名。
- `.轮盘 开始` / `.rr start`：房主提前开始多人轮盘。
- `.轮盘 扳机` / `.rr fire`：多人局当前行动者扣动道具扳机。安全者进入队尾，失败者出局。
- 多人局最后幸存者获得全部奖池。
- `.轮盘 状态` / `.rr status`：查看自己的单人局和本群多人局。
- `.轮盘 结束` / `.rr end`：结束自己的单人局，或由多人房主/管理员关闭多人局并退款。

## 21 点规则与命令

- `.21点 50` / `.bj 50`：下注并立即发牌。玩家拿两张明牌，系统庄家拿一张明牌和一张暗牌。
- `.bj hit` / `.21点 要牌`：当前手牌摸一张牌。
- `.bj stand` / `.21点 停牌`：当前手牌停牌。
- `.bj double` / `.21点 翻倍`：前两张牌时追加同额下注，补一张牌后停牌。
- `.bj split` / `.21点 分牌`：前两张点数相同时追加同额下注，拆成两手。分 A 后每手只补一张牌。
- `.bj insurance` / `.21点 保险`：庄家明牌为 A 时可买保险。
- `.bj surrender` / `.21点 投降`：前两张牌时放弃当前手牌，退回半注。
- `.bj status` / `.21点 状态`：查看自己的当前局。
- `.bj end` / `.21点 结束`：放弃当前局，按投降结算。
- 天然 Blackjack 只认初始两张牌 A 加 10 点牌，赔付 3:2。
- 普通胜利赔付 1:1，平局退回本金，玩家爆牌直接输掉本手下注。
- 系统庄家小于 17 点必须要牌，达到 17 点停牌。软 17 默认停牌，可在配置里改为要牌。

## 存储概要

- `wallet:<userId>`：钱包余额
- `debt:<userId>`：欠债
- `daily:<YYYY-MM-DD>:<userId>`：每日低保次数
- `walletUsers`：钱包索引
- `walletName:<userId>`：钱包展示名
- `jackpotWeight:<scopeId>`：老虎机大奖额外概率
- `jackpotPool:<scopeId>`：老虎机群奖池
- `fashion:<userId>`：时髦值
- `publicFund`：一键共产公款
- `rouletteSolo:<groupId|private:userId>:<userId>`：单人道具轮盘局
- `rouletteMulti:<groupId>`：多人道具轮盘局
- `blackjackRound:v2:<groupId>:<userId>`：21 点个人局
